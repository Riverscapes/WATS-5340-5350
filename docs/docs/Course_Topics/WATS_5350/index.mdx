## WATS 5350 Topics

Students learn the application of principles, theory, and skills to real-world aquatic ecosystem management and restoration projects. Assignments and group projects include diagnosing aquatic ecosystem impairments, objective definition, requests for proposals, restoration planning, regulatory permitting, and conceptual design. Field trips are required.

Topics include:
- [Geomorphic Principles](/Course_Topics/WATS_5350/Design_Geomorphic_Principles) - Module 1
- [Traditional Restoration Project Design - Grading Plans](/Course_Topics/WATS_5350/Grading) - Module 2
- [Conservation Finance & Resilience Bonds](/Course_Topics/WATS_5350/ConsFinance) - Module 3
- [Low-Tech Process-Based Restoration](/Course_Topics/WATS_5350/Low-Tech)
  - [Intro](/Course_Topics/WATS_5350/Low-Tech/intro) - Module 4
  - [Your Project](/Course_Topics/WATS_5350/Low-Tech/Projects/birch) - Module 5
  - [Planning](/Course_Topics/WATS_5350/Low-Tech/planning) - Module 6
  - [Design](/Course_Topics/WATS_5350/Low-Tech/design) - Module 7
  - [Implementation](/Course_Topics/WATS_5350/Low-Tech/) - Module 8
  - [Adaptive Management](/Course_Topics/WATS_5350/Low-Tech/adaptive) - Module 9
